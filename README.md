# Список дел
Приложение список дел на typeScript

## Функционал
1) Вывод спписка задач,
2) Добавление новой задачи,
3) Поиск по подстроке,
4) Завершение задачи,
5) Удаление задачи.

## Для реализации проекта использовалось
html,
scss,
js,
typeScript,
redux,
react-redux,
redux-thunk,
normalize.css,
prettier.

## Запуск
Приложение запускается на 3000 порте.

1 команда
### npm run build
2 команда
### npm run start
